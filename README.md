# battery-backup-prediction-ml# 🔋 Battery Backup Prediction using Machine Learning

## 🚀 Overview

This project demonstrates a simple Machine Learning model built using **Linear Regression** to predict mobile battery backup time based on battery capacity (mAh).

It is part of my learning journey in Machine Learning and focuses on understanding core concepts like model training, prediction, and parameter interpretation.

---

## 📊 Problem Statement

To analyze how battery capacity affects battery backup time and build a model that can predict backup duration for given battery specifications.

---

## 🧠 Model Used

* Linear Regression

---

## 🛠️ Technologies & Libraries

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib (optional for visualization)

---

## ⚙️ How It Works

1. Load and preprocess the dataset
2. Train the model using `model.fit()`
3. Predict battery backup using `model.predict()`
4. Analyze model parameters (slope & bias)

---

* Interpreting slope (w) and bias (b)
